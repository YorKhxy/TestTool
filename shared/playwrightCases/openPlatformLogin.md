# 开放平台管理端 - 登录功能测试用例

> 目标系统：http://192.168.1.73:3303
> 测试账号：13800000000 / 123456

## 用例格式说明

```typescript
import { test, expect } from '@playwright/test';

test('用例名称', async ({ page }) => {
  // 测试步骤
});
```

---

## TC001 - 页面基本元素验证

**目的**：验证登录页面所有基本元素正常显示

**前置条件**：访问登录页面

**测试步骤**：
1. 访问 http://192.168.1.73:3303/login
2. 等待页面加载完成
3. 验证页面标题包含"开放平台"、"登录"或"管理"
4. 验证手机号输入框可见
5. 验证密码输入框可见
6. 验证登录按钮可见

**预期结果**：
- [ ] 页面标题正确
- [ ] 手机号输入框可见
- [ ] 密码输入框可见
- [ ] 登录按钮可见

**参考代码**：
```typescript
test('页面基本元素验证', async ({ page }) => {
  await expect(page).toHaveTitle(/开放平台|登录|管理/);
  await expect(page.locator('手机号输入框')).toBeVisible();
  await expect(page.locator('密码输入框')).toBeVisible();
  await expect(page.locator('登录按钮')).toBeVisible();
});
```

## TC002 - 正常登录流程

**目的**：验证使用正确账号密码能够成功登录

**前置条件**：访问登录页面

**测试数据**：
- 手机号：13800000000
- 密码：123456

**测试步骤**：
1. 输入手机号：13800000000
2. 输入密码：123456
3. 点击登录按钮
4. 等待页面跳转

**预期结果**：
- [ ] 登录后跳转到 Dashboard/首页（URL 不再包含 /login）
- [ ] 页面显示欢迎信息或主控台内容

**参考代码**：
```typescript
test('正常登录流程 - 成功登录', async ({ page }) => {
  await page.locator('手机号输入框').fill('13800000000');
  await page.locator('密码输入框').fill('123456');
  await page.locator('登录按钮').click();
  await page.waitForURL('**/dashboard**', { timeout: 10000 });
  expect(page.url()).not.toContain('/login');
});
```
